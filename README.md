# Digital Service for Quality of Life Improvement

## Описание

Проект направлен на создание цифрового сервиса для улучшения качества жизни. Он включает несколько трекеров и менеджеров, которые помогают пользователям следить за своим здоровьем, финансами, снами и физической активностью. Этот сервис создан с использованием современных технологий, таких как Laravel, Vue.js, Tailwind CSS и Docker, с фокусом на модульную архитектуру и лучшие практики разработки.

## Стек технологий

### Backend
- **Laravel 10** (PHP > 8.2)
- **Swagger L5** — для документации API
- **Docker** — для контейнеризации приложения
- **Postman** — для тестирования API
- **PHPStorm** — для разработки

### Frontend
- **Vue.js 3** (Composition API)
- **Tailwind CSS** — для стилизации
- **WebStorm** — для разработки

### Принципы разработки
- **DRY** (Don't Repeat Yourself)
- **SOLID**
- **CQRS** (Command Query Responsibility Segregation)
- **YAGNI** (You Aren't Gonna Need It)
- **KISS** (Keep It Simple, Stupid)

### Инструменты CI/CD
- **GitHub Actions** — для автоматизации процессов
- **GitHub** — для хостинга кода и управления версиями

## Структура проекта

Проект разделен на несколько папок:

- **`backend/`** — серверная часть на Laravel
- **`frontend/`** — клиентская часть на Vue.js
- **`docker/`** — конфигурации для Docker

## Функциональность

### 1. **Таск менеджер**
Позволяет пользователю добавлять, редактировать, удалять задачи и сортировать их по различным параметрам. Также есть возможность выполнения задач с возможностью отслеживания их статуса.

#### Особенности:
- Добавление, редактирование и удаление задач
- Сортировка по приоритету, сроку и статусу
- Выполнение задач с отслеживанием прогресса

### 2. **Финансовый трекер**
Позволяет пользователю отслеживать свои финансы, транзакции через PDF транскрипт от Каспи и получать советы на основе их зарплаты. Включает систему целей для накоплений.

#### Особенности:
- Загрузка и обработка PDF транскриптов из Каспи
- Советы по финансовому планированию на основе зарплаты
- Установка и отслеживание сберегательных целей

### 3. **Трекер сна**
Позволяет отслеживать режим сна, записывать время засыпания и пробуждения, а также анализировать качество сна.

#### Особенности:
- Запись времени засыпания и пробуждения
- Анализ продолжительности сна
- Рекомендации по улучшению качества сна

### 4. **Трекер воды**
Позволяет пользователю отслеживать потребление воды на протяжении дня и напоминать ему о необходимости пить воду.

#### Особенности:
- Отслеживание потребления воды
- Напоминания по питьевому режиму

### 5. **Трекер спорта**
Позволяет пользователю отслеживать физическую активность, тренировки и упражнения. Пользователи могут создавать собственные тренировки и отслеживать их выполнение.

#### Особенности:
- Шаблоны для различных типов упражнений
- Возможность создавать свои тренировки
- Статистика и отчеты о выполнении

### 6. **Трекер калорий и диеты**
Позволяет пользователям отслеживать потребление калорий, составлять диеты и следить за своим рационом.

#### Особенности:
- Подсчет калорий в потребляемой пище
- Создание и отслеживание диетических планов
- Статистика по калориям и макронутриентам

## Установка

### Требования
- PHP >= 8.2
- Composer
- Node.js и NPM
- Docker
- MySQL

### Установка зависимостей

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourusername/repository-name.git
2. Перейдите в папку проекта:
   ```bash
   cd repository-name
3. Установите зависимости для backend:
   ```bash
   cd backend
   composer install
4. Установите зависимости для frontend:
   ```bash
   cd frontend
   npm install
5. Настройте Docker (если используете Docker для контейнеризации):
   ```bash
   docker-compose up --build
6. Настройте переменные окружения для Laravel:
   ```bash
   cp .env.example .env
   php artisan key:generate
   php artisan migrate
7. Запустите фронтенд:
   ```bash
   cd frontend
   npm run dev
