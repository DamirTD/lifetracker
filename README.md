## ⚙️ Установка

### Предварительные требования
- PHP ≥ 8.2
- Node.js ≥ 18.x
- Docker и docker-compose
- Composer 2.x

### Пошаговая установка

1. Клонирование репозитория:
```bash
git clone https://github.com/yourusername/repository-name.git
cd repository-name
```
2. Настройка бэкенда:
```
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
```

3. Запуск фронтенда:
```
cd ../frontend
npm install
npm run dev
```

4. (Опционально) Docker-развёртывание:
```
docker-compose up --build -d
```
